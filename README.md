# Gold Calculator

Modern Next.js scrap **gold calculator** with live spot pricing — built to outperform outdated form-based tools.

## Features

- Instant melt-value estimates (no page reload)
- Multi-piece jewelry lots
- Grams, mg, DWT, grains, oz, troy oz
- Live USD / troy oz spot + manual override
- Typical buyer offer ranges (70% / 85% / 95%)
- Karat purity guide + FAQ for SEO

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
